Redstone IC Project



Redstone Wire

Version 1

Note: Redstone wire doesn't function exactly How it does in game but shows it main feature of signal propogation around the wire

Description

Shows redstone components signal traveling across the wire and stays on for as long as the signal is on.

Parts Used

  • Resistors(x5)(100-220)
  • Leds(x5)
  • Transistor NPN
  • Transistor PNP

How It Works

Led voltage is sourced through the power supply rather than the signal and has the signal activate the transistors to allow positive voltage to flow through Leds to turn them on, but turns of as soon as the signal goes to low.
